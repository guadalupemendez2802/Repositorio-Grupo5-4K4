Desarrollamos la UserStorie Comprar entradas
Como visitante quiero comprar entradas para poder ingresar al parque
C.A.:
    - Se debe ingresar fecha de visita deseada, cantidad de entradas y edad de cada visitante
    - Se debe ingresar medio de pago
    - Se debe enviar mail de confirmacion y entradas
P.U.:
    - Se ingresa una fecha anterior a la actual (FALLA)
    - Se seleccionan mas de 10 entradas (FALLA)
    - No se selecciona ningun medio de pago (FALLA)
    - Probar comprar entrada ingresando fecha futura, cantidad mayor a 0 y menor a 10, seleccionando forma de pago valida y enviando mail (PASA)

*Para las pruebas de usuario de FALLA se puede hacer una completa o cada una por su parte (como esta hecho aca)
*Para las pruebas de usuario de PASA se debe hacer completa (como esta descrita aca) porque si fuera una por una no pasaria
